# 🔎 도넛과 막대 그래프

https://school.programmers.co.kr/learn/courses/30/lessons/258711

## 💡 아이디어

- dict에 각 노드의 출발과 도착을 담아둔다.
- 도넛, 막대, 8자 그래프의 특징을 파악하여 갯수를 구한다.

## ✔ 문제풀이

- graph관련 dict를 생성한다.
- edges를 돌면서 key에 노드들을 담고 value에는 [출발간선갯수, 도착간선갯수]를 담는다.
- key값을 순회하면서 각 조건에 맞는 경우에 answer값을 변경한다.
  > 1. 시작 정점 : 도착간선은 없고 출발 간선만 2개 이상이어야 한다.
  > 2. 막대 모양 : 출발간선은 없고 도착 간선만 1개 이상이다. (마지막에 있는 노드를 생각하면 된다.)
  > 3. 8자 모양 : 8자의 가운데를 생각했을 때 출발과 도착이 각각 2개 이상이다.
  > 4. 도넛 모양 : 막대와 8자를 제대로 구했다면 시작 정점에서 출발한 간선에서 다른 모양들의 갯수를 제거하면 도넛 모양이다.

## 🤕 어려웠던 점

- 담는 것까지는 어렵지 않았지만 어떤 경우에 그래프들이 맞는지 어려웠다.. 막대제외하고 8자와 도넛을 몰라서 답을 보고 알았다.
- 구현이 어렵다기보다는 창의력문제의 느낌이었다.
