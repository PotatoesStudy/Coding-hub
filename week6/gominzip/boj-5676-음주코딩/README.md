# 🔎 문제이름

boj 5676 음주코딩

## 💡 아이디어

- 세그먼트 트리. 구간 곱

## ✔ 문제풀이

- 세그먼트 트리 만들기
  - 왼쪽 절반 구간의 구간곱과 오른쪽 절반 구간의 구간곱을 곱해 현재 구간곱으로 취하기
- 이때 해당 문제는 0,양수,음수만 판단하면 되므로 convert를 활용해서 부호만 취했음
- 트리를 생성하는 build, 변경값으로 변환해서 결과 변경하는 update, 특정 구간의 결과를 계산해서 반환하는 query

## 🤕 어려웠던 점

- 구간 합은 예전에 풀었는데 세그먼트 트리 방식은 이번에 처음 접함
- 비슷한 유형도 풀어보기
