prgs 점찍기

## 💡 아이디어

- 그냥 구현?

- 이후 반복문으로 a와 b 값을 대입하고, d를 넘는 경우에는 break;

- 1차 시도에서 위처럼 생각했는데 효율성 이슈

- k가 곱해진 걸 놓쳤다.
