  - q의 0번 인덱스의 시간이 i번째와 똑같으면 빼고 plus_server에 q의 0번 인덱스에 있는 서버만큼 빼준다.
  - 해당 시간의 player가 m보다 큰 경우 필요한 서버갯수를 구하고 만약 추가된 서버보다 많을 경우 answer에 더 추가해야하는 서버만큼 더한다.

  - plus_server는 그 다음에 추가해준다. -> **먼저 해버리면 q가 꼬여서 이상한 답이 나온다.**
