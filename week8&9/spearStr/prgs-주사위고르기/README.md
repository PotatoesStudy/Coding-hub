## ✔ 문제풀이
- 인덱스를 하나씩 올리면서 주사위를 `2 / n`개 선택한다.
    - 다 선택했다면, 승수를 파악해야한다. totalWin 함수로 파악한다.
    - getSums 함수를 통해 모든 조합의 합들을 key와 value를 가진 자료구조로 구한다.
    - key값을 순회하면서 이긴수를 더하고 이긴수가 최대가 될때의 조합을 저장한다.
